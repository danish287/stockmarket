# StockMarket Application
***

This is an application that displays daily historical time series for a digital currency (e.g., BTC) traded on a specific market (e.g., USD), refreshed daily at midnight (UTC). Prices and volumes are quoted in both the market-specific currency and USD. 


## Table of Contents
1. [General Info](#general-info)
2. [Technologies](#technologies)

### General Info
***
Project Status: in progress.

This application is powered by Alpha Vantage which provides enterprise-grade financial market data through a set of powerful and developer-friendly APIs.
